### 设置代理

```shell
# CMD
set http_proxy=http://127.0.0.1:0 & set https_proxy=http://127.0.0.1:0
# npm
$ npm config set HTTPS_PROXY=http://127.0.0.1:6152
$ npm config set HTTP_PROXY=http://127.0.0.1:6152
```
